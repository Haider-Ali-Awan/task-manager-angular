About Task Manager

This Task Manager project is built with Angular, Angular Material, and Bootstrap 5.

  • Dashboard with task summary cards
  • Task list with Material cards
  • Add Task form with reactive validations
  • Edit/Delete functionality via Angular Material dialogs
  • Success/Error notifications with Material snackbars

SCSS is used for styling dialogs, cards, and snackbars, while Bootstrap ensures a responsive layout.

Note: Tasks are not persisted on refresh; you can integrate localStorage or a backend for data persistence.
